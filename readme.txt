All-In-One Login Pack is a collection of 4 components which help to make login & registration forms work with your theme.

== Description ==
WordPress login & registration are built-in functionality. After installation, you can enable registration in the admin area. However, these are not part of the theme package. Making customization is limited and more difficult.

All-In-One Login Pack is a collection of 4 components which help to make login & registration forms work with your theme. You can use the shortcodes and integrate them with any page url. All of the forms are part of your theme by using shortcodes.

Features

   * Use shortcodes to put forms in your pages, widgets and any other areas.
   * Registration supports terms & conditions page and confirmation email.
   * Use PHP validation, no JavaScript files to slow your site.

Read more details about this <a href="http://gidd.org/wordpress-plugins/login-pack/" target="_blank">login pack</a>, including demo site & how to use.

== Installation ==
1.  Upload All-In-One Login Pack to the /wp-content/plugins/ directory.
2.  Activate the plugin through the Plugins menu item in the WordPress Dashboard.
3.  Once the plugin is activated, configure page IDs in settings and use shortcodes to display.
